データ構造

キー型内容
metaObjectデータセットのメタ情報（タイトル・年度・バージョン）
population.studentsNumber受験者総数（124,583 人）
scoreRangeObject得点範囲 min / max
statisticsObject平均・中央値・最頻値・分散・標準偏差・四分位数
percentilesObjectp10 〜 p99 パーセンタイル値
questionAveragesArray問1〜問10 各問の平均点
distributionArray得点帯ごとの度数（score / count）
gradingGuideObject評価区分のしきい値（優 / 良 / 可 / 要改善）

コードサンプル

// fetch API でデータを取得する（コールバック形式）
const URL = "https://yuu-miino-nue.github.io/2026_ICT_lecture/data/math-statistics-2025.json";

fetch(URL)
  .then(res => res.json())
  .then(data => {
    console.log("平均点:", data.statistics.mean);
    console.log("標準偏差:", data.statistics.standardDeviation);
    console.log("受験者数:", data.population.students);
  })
  .catch(err => console.error("Error:", err));
// async/await 形式
const URL = "https://yuu-miino-nue.github.io/2026_ICT_lecture/data/math-statistics-2025.json";

async function loadStats() {
  const res  = await fetch(URL);
  const data = await res.json();

  // 自分の点数を入力して順位を推定する例
  const myScore = 75;
  const mean = data.statistics.mean;
  const sd   = data.statistics.standardDeviation;
  const z    = (myScore - mean) / sd;

  console.log(`z スコア: ${z.toFixed(2)}`);
  return data;
}

loadStats();
import urllib.request
import json

URL = "https://yuu-miino-nue.github.io/2026_ICT_lecture/data/math-statistics-2025.json"

with urllib.request.urlopen(URL) as res:
    data = json.loads(res.read())

print("平均点:", data["statistics"]["mean"])
print("標準偏差:", data["statistics"]["standardDeviation"])

# 度数分布を表示
for entry in data["distribution"]:
    bar = "█" * (entry["count"] // 1000)
    print(f"{entry['score']:3d} 点台: {bar} ({entry['count']:,}人)")
library(jsonlite)

URL <- "https://yuu-miino-nue.github.io/2026_ICT_lecture/data/math-statistics-2025.json"
data <- fromJSON(URL)

# 基本統計量の表示
cat("平均:", data$statistics$mean, "\n")
cat("標準偏差:", data$statistics$standardDeviation, "\n")

# 度数分布のプロット
dist <- data$distribution
barplot(dist$count,
        names.arg = dist$score,
        main = "得点分布",
        xlab = "得点帯",
        ylab = "人数",
        col  = "steelblue")

注意事項

このデータはフィクションです。 教育目的で作成したモックデータであり、実際の全国試験結果とは一切関係ありません。
  • データは読み取り専用です（PUT / POST / DELETE は使用できません）。
  • GitHub Pages は静的ファイルホスティングのため、フィルタリングやクエリパラメータは使用できません。
  • JSON 全体を取得してから JavaScript/Python 側で加工してください。
  • リクエスト頻度の制限はありませんが、演習中の不必要な連続アクセスは避けてください。
